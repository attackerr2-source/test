# 📊 دليل تخزين البيانات والإعدادات

## 🗂️ موقع البيانات

جميع البيانات والإعدادات محفوظة في قاعدة بيانات SQLite:

```
📁 bot-main/
  └── 📁 db/
      ├── 📄 bot_data.db         ← **قاعدة البيانات الرئيسية** (96 KB)
      ├── 📄 database.py         ← معالج قاعدة البيانات
      └── 📁 db_backups/         ← النسخ الاحتياطية التلقائية
```

---

## 📝 أنواع البيانات المحفوظة

### 1️⃣ **إعدادات البوت** (لا تُحذف عند الإعادة)
```
botmak/[BOT_ID]/setting
├── type              → تفعيل استقبال الرسائل (✅/❌)
├── replymod          → وضع الرد (✅/❌)
├── modetext1-10      → تحكم الوسائط ✅ مسموح / ❌ ممنوع
│   ├── modetext1   → الصور
│   ├── modetext2   → الموسيقى
│   ├── modetext3   → الملفات
│   ├── modetext4   → الملصقات
│   ├── modetext5   → الفيديو
│   ├── modetext6   → الصوتيات
│   ├── modetext7   → جهات الاتصال
│   ├── modetext8   → إعادة التوجيه
│   ├── modetext9   → جميع الروابط
│   └── modetext10  → روابط تيلجرام
└── notif_*          → الإخطارات
```

### 2️⃣ **بيانات تشغيلية**
```
botmak/[BOT_ID]/
├── token             → توكن البوت
├── admin             → معرف الأدمن الرئيسي
├── allUser           → قائمة المستخدمين
├── zune              → إعدادات إضافية (قنوات، رسائل ترحيب)
├── count/user/[ID]   → عدد الرسائل من كل مستخدم
├── count/user/all    → العدد الكلي للرسائل
└── NaMero/
    ├── ban           → قائمة المحظورين
    └── member        → قائمة الأعضاء
```

## 🔍 أدوات للوصول للبيانات

### 1. عرض جميع البيانات
```bash
python view_database.py
```

### 2. البحث عن إعداد معين
```bash
python view_database.py search setting
python view_database.py search modetext4
```

### 3. تصدير النسخة الاحتياطية
```bash
python view_database.py export backup.json
```

### 4. برنامج النسخ الاحتياطي التلقائي
```bash
python backup_database.py
```

---

## ✅ التحقق من حفظ البيانات

عند تشغيل البوت:
1. البيانات تُحفظ **فوراً** في `db/bot_data.db`
2. عند إعادة التشغيل، جميع البيانات **تبقى محفوظة**
3. الإعدادات لا تُحذف أبداً إلا بحذف `db_data.db` يدويّاً
